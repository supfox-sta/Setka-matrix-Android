# Сетка Matrix Android

Мобильный клиент «Сетка Matrix Android» — открытая Android-версия, созданная вашей командой для доступа к домашней инстанции Matrix на `setka-matrix.ru`.

Проект построен поверх Element X и наследует от него все ключевые механизмы взаимодействия с серверами Matrix, включая поддержку расписаний комнат, шифрование и работу с медиа. Ваша команда добавляет кастомные настройки темы, фирменные стили и преднастроенные серверы для быстрого подключения к Сети Matrix.

Команда делает акцент на безопасной коммуникации, управлении уведомлениями и встроенной справке, чтобы пользователи могли сразу войти в ваш сервер без долгой настройки.
